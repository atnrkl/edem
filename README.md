## Installation

E-democracy requires [Node.js](https://nodejs.org/) v10+ to run.

Install the dependencies and devDependencies and start the server.

```sh
git clone https://github.com/atnrkl/edem
cd edem
npm i
npm run dev
```

open up

```sh
localhost:3000
```

in your browser
